My name is Michael. I am a graduate student studying (Computational) Linguistics at BYU. I hope you benefit from some of my projects. This site constitutes a collection of original and derivative work. Where appropriate, I provide a link to the author's work.

My Projects and Utilities

  • Ling Utils Project: My favorite linguistic utilities listed in one place. I have made local copies when possible so even if a webserver turns off, the utility is still available.
  • WP SCI Windows Phone 7, 8 version of the Scripture Citation Index (Online Versions: http://scriptures.byu.edu/, http://scriptures.byu.edu/sci2/, http://scriptures.byu.edu/mapscrip/)
  • BYU Testing Center Project (also available in the google chrome store for free)
  • BYU Webcams Project (also available in the google chrome store for free). Shows you the weather outside and inside. It also automatically updates.
  • Random Talk (may soon be available in the google chrome store for free). Displays a random LDS conference talk. Great for those moments when you need a modern spiritual pick-me-up! (Retry page if you get an error)
  • Random Tree Drawer Generates a random tree every time you visit. You can watch it build the branches as it goes! Great reminder to do some family history research.
  • LaTeX Project Management Template An ongoing LaTeX project that provides the core project management paperwork. This includes stakeholder tables and more.
  • Web Transformer A step-by-step demonstration of the process through which a web page might undergo until it reaches web nirvana. Just keep clicking "Navigate to next" when you are ready to see what transformations remain.
